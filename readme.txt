Allows you to limit the amount of words in a comment.

== Description ==
Comments can often get out of hand. Limit the amount of words with this plugin and get more concise conversations in your comments.

== Installation ==

= Admin Installer via search =
1. Visit the Add New plugin screen and search for "comment word limiter".
2. Click the "Install Now" button.
3. Activate the plugin.
4. A new setting will be added to the Discussion page

= Admin Installer via zip =
1. Visit the Add New plugin screen and click the "Upload Plugin" button.
2. Click the "Browse..." button and select zip file from your computer.
3. Click "Install Now" button.
4. Once done uploading, activate Comment Word Limiter.

= Manual =
1. Upload the Comment Word Limiter folder to the plugins directory in your WordPress installation.
2. Activate the plugin.
3. A new setting will be added to the Discussion page

That's it!
